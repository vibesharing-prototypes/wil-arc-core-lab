# Global instances layout (archive)

Frozen **information architecture** for the workflows home surface: a **single “Active instances” block** below the **workflow templates** list, shared across templates rather than nested per template.

- **Why it exists:** Preserved in the lab for stakeholder discussion and comparison with the **per-template** layout used on `custom-attributes-v2` `/workflows`.
- **Route:** `/explorations/workflows-global-instances-archive` (`WorkflowsGlobalInstancesArchivePage.tsx`).
- **Source of truth for the current product direction:** `custom-attributes-v2` — instances are scoped under each template card.
