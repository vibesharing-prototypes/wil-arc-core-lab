# M0 Decisions (Attribute Renderer)

- Boolean: render as **Yes/No text** in read-only (no toggle UI).
- Currency: support both; default examples use **perAttribute (USD)**.
- Attachment: support **multiple**; read-only shows a simple list.
- Selects: if option is deprecated, append **"(Deprecated)"** in read-only.
- Description placement: **inline helper text** below label; optional tooltip for long guidance.
